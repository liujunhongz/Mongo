## 用法

### 1. 读取excel文件

```
java -jar mongo.jar -import file
```
### 2. 导出mongo数据到excel表中

```
java -jar mongo.jar [-export]
```
### 3. 查看帮助

```
java -jar mongo.jar -h
```
